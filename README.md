## Installation

1. **Clone the repository**
   ```sh
   git clone https://github.com/yourusername/modern-task-manager.git
   cd modern-task-manager
   ```

2. **Install dependencies**
   ```sh
   flutter pub get
   ```

3. **Run the app**
   ```sh
   flutter run
   ```
